## Installation 🛠️

1. **Clone the Repository** 📥:
 ```bash
 git clone https://github.com/username/Gemini-ChatWithPDFs.git
 cd Gemini-ChatWithPDFs
 ```

2. **Install Dependencies** 📦:
 ```bash
 pip install -r requirements.txt
 ```
